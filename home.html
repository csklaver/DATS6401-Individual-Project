Welcome to my Visualization of Complex Data Project I Home Page

This projet explores the healthcare, education, and military spending of some of the top countries in the world. This includes 15 of the G-20 countries from 2013 to 2018.
Countries were chosen given that data was avaiable from at least 2015. The data was retrieved from the World Bank DataBank.
Missing data on education spending was supplemented from the OECD and tradingeconomics.com. Other missing values were imputed with averages.
Please explore the visuals in each section including an overview of spending in each sector, comparisons of each type of spending, and trends over time.
A link to my GitHub page including all code and data is provided in the footer.
